About the Project

This website was created as a class project for IT 3203 - Intro to Web Development. The project aims to provide an overview of HTTP and its evolution, covering topics such as the history of HTTP, how it works, and modern implementations like HTTP/2 and HTTP/3. The website is structured with a navigation menu that allows users to easily access different sections of the content, making it a comprehensive resource for anyone interested in learning about HTTP and its importance. The project also includes references to various sources used in the creation of the content, ensuring that users can further explore the topics discussed on the website.
